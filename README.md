# Spatial Clustering of Infectious Syphilis in Victoria, Australia

Replication code and data for the spatial analysis of infectious syphilis notification patterns across Victorian local government areas (LGAs), 2019-2024. This analysis applies spatial autocorrelation, spatial regression, geographically weighted regression, and standardised morbidity ratio decomposition to examine the relationship between MSM residential geography and syphilis notification clustering.

**Author:** Hayden Farquhar ([ORCID: 0009-0002-6226-440X](https://orcid.org/0009-0002-6226-440X))

**Preprint:** [doi.org/10.64898/2026.02.13.26346302](https://doi.org/10.64898/2026.02.13.26346302)

**Code archive:** [![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19564000.svg)](https://doi.org/10.5281/zenodo.19564000)

## Overview

This analysis uses publicly available aggregate surveillance data from the Victorian Department of Health, ABS Census 2021, and the National Health Services Directory. The study examines:

1. **Spatial autocorrelation** (Moran's I, LISA, Getis-Ord Gi*) and temporal persistence of clustering
2. **Spatial regression** comparing Census same-sex couple density against a demographic proxy (% males 20-44) as area-level MSM population measures
3. **Geographically weighted regression** (GWR) to identify spatially varying associations
4. **MSM population estimation** using Wilson and Shalley's Census-based distribution method, validated against three external benchmarks
5. **SMR decomposition** to disentangle demographic concentration from per-MSM epidemiological heterogeneity
6. **Sensitivity analyses** including weight sensitivity, leave-one-out, scaling invariance, proxy comparison, and 2016-2021 Census temporal stability

## Data Acquisition

### Included in this repository

The following small data files are included directly in `data/raw/`:

- **Kirby Institute national data** (8 CSVs): Notification counts/rates by age, sex, Indigenous status, remoteness, and state
- **VIC Health** (6 CSVs): Syphilis notifications by LGA, 2019-2024
- **NHSD services** (1 CSV): Sexual health clinic locations

### Manual downloads required

The following files must be downloaded manually before running the analysis.

#### 1. ABS Geographic Boundaries → `data/raw/boundaries/`

Download ASGS 2021 Edition shapefiles (GDA2020 CRS) from the [ABS ASGS page](https://www.abs.gov.au/statistics/standards/australian-statistical-geography-standard-asgs-edition-3/jul2021-jun2026/access-and-downloads/digital-boundary-files):

| File | Expected path after extraction |
|------|-------------------------------|
| SA2 2021 shapefile ZIP | `data/raw/boundaries/SA2_2021/SA2_2021_AUST_GDA2020.shp` |
| LGA 2021 shapefile ZIP | `data/raw/boundaries/LGA_2021/LGA_2021_AUST_GDA2020.shp` |
| STE 2021 shapefile ZIP | `data/raw/boundaries/STE_2021/STE_2021_AUST_GDA2020.shp` |

#### 2. ABS Census 2021 → `data/raw/demographics/`

| File | Source |
|------|--------|
| `Census_2021_G07_Indigenous_SA2.zip` | [ABS Census DataPacks](https://www.abs.gov.au/census/find-census-data/datapacks) — General Community Profile at SA2 level (contains G01 total population and G07) |
| `SEIFA_2021_SA2.xlsx` | [ABS SEIFA](https://www.abs.gov.au/statistics/people/people-and-communities/socio-economic-indexes-areas-seifa-australia/2021) — Table 1, IRSD by SA2 |
| `RA_2021_AUST.xlsx` | [ABS Remoteness Areas](https://www.abs.gov.au/statistics/standards/australian-statistical-geography-standard-asgs-edition-3/jul2021-jun2026/access-and-downloads/remoteness-structure) — SA1-level remoteness classification |
| `ERP_SA2_2001-2024.xlsx` | [ABS ERP](https://www.abs.gov.au/statistics/people/population/regional-population/latest-release) — Table 1, ERP by LGA (note: file is LGA-level despite filename) |

#### 3. ABS Same-Sex Couple Data → `data/raw/demographics/`

Obtained from [ABS TableBuilder](https://www.abs.gov.au/statistics/microdata-tablebuilder/tablebuilder) (free registration required):

| File | TableBuilder specification |
|------|--------------------------|
| `ABS_SSCF_VIC_LGA_2021.csv` | 2021 Census > Selected family characteristics > Counting: Families > Row: LGA (Victoria) > Column: SSCF Same-Sex Couple Indicator |
| `ABS_SSCF_VIC_LGA_2016.csv` | Same specification using 2016 Census dataset (for temporal stability validation) |

## Replication Workflow

After downloading all required data files, run the scripts in order:

```r
setwd("/path/to/this/repository")

source("R/00_setup.R")               # Load packages and paths
source("R/01_data_acquisition.R")     # Process raw data → RDS
source("R/02_data_cleaning.R")        # Build analysis datasets
source("R/03_descriptive_maps.R")     # Descriptive figures
source("R/04_spatial_autocorrelation.R")  # Spatial analysis + regression
source("R/08_manuscript_figures.R")   # Publication figures and tables
source("R/09_sensitivity_diagnostics.R")  # Diagnostics and validation
```

### Expected run time

The full pipeline takes approximately 10-15 minutes on a standard laptop. The spatial analysis (script 04) and sensitivity diagnostics (script 09) are the most computationally intensive steps.

### Output

- `data/processed/` — RDS files containing cleaned, analysis-ready datasets
- `output/figures/` — Trend plots, scatter plots, funnel plots, diagnostic plots (PDF + PNG)
- `output/maps/` — Choropleth maps, LISA cluster maps, Gi* hotspot maps, GWR maps (PDF + PNG)
- `output/tables/` — Regression results, SMR estimates, proxy comparisons, model diagnostics (HTML + CSV)
