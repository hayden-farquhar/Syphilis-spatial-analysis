# Spatial and Temporal Analysis of Australia's Syphilis Surge

Replication code and data for the spatial and temporal analysis of infectious syphilis notifications across Australia (2011-2024). This repository accompanies the manuscript examining geographic clustering patterns, demographic disparities, congenital syphilis risk factors, and the impact of COVID-19 on syphilis notification trends.

**Author:** Hayden Farquhar ([ORCID: 0009-0002-3399-6428](https://orcid.org/0009-0002-3399-6428))

## Overview

This analysis uses publicly available aggregate surveillance data from Australian state and territory health departments, the Kirby Institute, and the Australian Bureau of Statistics. The study examines:

1. **Descriptive epidemiology** of the syphilis surge by jurisdiction, demographics, and remoteness
2. **Spatial autocorrelation** (Moran's I, LISA, Getis-Ord Gi*) and spatial regression at the LGA level
3. **Geographically weighted regression** (GWR) to identify spatially varying risk factors
4. **Congenital syphilis** trends and composite risk mapping at the SA3 level
5. **COVID-19 impact** on notification rates using interrupted time series analysis
6. **Sensitivity analyses** including NSW monthly ITS (n=204 observations) and model diagnostics

## Data Acquisition

### Included in this repository

The following small data files are included directly in `data/raw/`:

- **Kirby Institute national data** (8 CSVs): Notification counts/rates by age, sex, Indigenous status, remoteness, and state
- **QLD Health** (13 CSVs): Infectious and congenital syphilis by HHS, age, sex, Indigenous status, region
- **VIC Health** (6 CSVs): Syphilis notifications by LGA, 2019-2024
- **NT Health** (4 CSVs): Infectious and congenital syphilis by district/quarter
- **WA Health** (1 CSV): SORG communique summary
- **NSW Health** (1 CSV): Monthly infectious syphilis notifications, 2009-2025
- **NHSD services** (1 CSV): Sexual health and Aboriginal health service locations

### Manual downloads required

The following files must be downloaded manually before running the analysis. Place each file in the specified directory.

#### 1. ABS Geographic Boundaries → `data/raw/boundaries/`

Download ASGS 2021 Edition shapefiles (GDA2020 CRS) from the [ABS ASGS page](https://www.abs.gov.au/statistics/standards/australian-statistical-geography-standard-asgs-edition-3/jul2021-jun2026/access-and-downloads/digital-boundary-files):

| File | Expected directory after extraction |
|------|-------------------------------------|
| SA2 2021 shapefile ZIP | `data/raw/boundaries/SA2_2021/SA2_2021_AUST_GDA2020.shp` |
| SA3 2021 shapefile ZIP | `data/raw/boundaries/SA3_2021/SA3_2021_AUST_GDA2020.shp` |
| LGA 2021 shapefile ZIP | `data/raw/boundaries/LGA_2021/LGA_2021_AUST_GDA2020.shp` |
| STE 2021 shapefile ZIP | `data/raw/boundaries/STE_2021/STE_2021_AUST_GDA2020.shp` |

Also download these GeoJSON boundary files:

- **PHN boundaries** (`PHN_boundaries.geojson`): From the [Australian Government data.gov.au](https://data.gov.au) — search "Primary Health Network boundaries"
- **QLD HHS boundaries** (`QLD_HHS_boundaries.geojson`): From the [QLD Open Data Portal](https://www.data.qld.gov.au) — search "Hospital and Health Service boundaries"

#### 2. ABS Census 2021 → `data/raw/demographics/`

| File | Source |
|------|--------|
| `Census_2021_G07_Indigenous_SA2.zip` | [ABS Census DataPacks](https://www.abs.gov.au/census/find-census-data/datapacks) — download "General Community Profile" at SA2 level. The ZIP contains G01 (total population) and G07 (Indigenous population) CSVs. |
| `SEIFA_2021_SA2.xlsx` | [ABS SEIFA](https://www.abs.gov.au/statistics/people/people-and-communities/socio-economic-indexes-areas-seifa-australia/2021) — Table 1, Index of Relative Socio-economic Disadvantage by SA2 |
| `RA_2021_AUST.xlsx` | [ABS Remoteness Areas](https://www.abs.gov.au/statistics/standards/australian-statistical-geography-standard-asgs-edition-3/jul2021-jun2026/access-and-downloads/remoteness-structure) — Remoteness Area classification at SA1 level |
| `ERP_SA2_2001-2024.xlsx` | [ABS Estimated Resident Population](https://www.abs.gov.au/statistics/people/population/regional-population/latest-release) — Table 1, ERP by LGA (note: file is LGA-level despite filename) |

#### 3. NSW Health XLS files → `data/raw/notifications/`

Download from [NSW Health Infectious Diseases Data](https://www.health.nsw.gov.au/Infectious/Pages/data.aspx):

| File | Description |

## Replication Workflow

After downloading all required data files, run the scripts in order from the project root directory:

```r
# Set working directory to repository root
setwd("/path/to/this/repository")

# Step 1: Load packages and configure paths
source("R/00_setup.R")

# Step 2: Read raw data and save standardised RDS objects
source("R/01_data_acquisition.R")

# Step 3: Harmonise notifications and build analysis datasets
source("R/02_data_cleaning.R")

# Step 4: Generate descriptive figures and choropleth maps
source("R/03_descriptive_maps.R")

# Step 5: Spatial autocorrelation, regression, and GWR
source("R/04_spatial_autocorrelation.R")

# Step 6: Congenital syphilis analysis and risk mapping
source("R/06_congenital_analysis.R")

# Step 7: COVID-19 interrupted time series (national, annual)
source("R/07_covid_impact.R")

# Step 8: NSW monthly ITS sensitivity analysis
source("R/07b_nsw_monthly_its.R")

# Step 9: Sensitivity analyses and model diagnostics
source("R/09_sensitivity_diagnostics.R")
```

Alternatively, run each script individually from the command line:

```bash
Rscript R/00_setup.R
Rscript R/01_data_acquisition.R
Rscript R/02_data_cleaning.R
Rscript R/03_descriptive_maps.R
Rscript R/04_spatial_autocorrelation.R
Rscript R/06_congenital_analysis.R
Rscript R/07_covid_impact.R
Rscript R/07b_nsw_monthly_its.R
Rscript R/09_sensitivity_diagnostics.R
```

### Expected run time

The full pipeline takes approximately 10-15 minutes on a standard laptop. The spatial analysis (script 04) and sensitivity diagnostics (script 09) are the most computationally intensive steps.

### Output

After running all scripts, the following directories will be populated:

- `data/processed/` — ~45 RDS files containing cleaned, analysis-ready datasets
- `output/figures/` — Temporal trend plots, demographic stratification charts (PDF + PNG)
- `output/maps/` — Choropleth maps, LISA cluster maps, Gi* hotspot maps, GWR coefficient maps (PDF + PNG)
- `output/tables/` — Summary statistics, regression results, model diagnostics (HTML + CSV)

## Coordinate Reference System

All spatial data use GDA2020 geographic coordinates (EPSG:7844), the Australian standard geodetic datum.

## Ethical Considerations

- This analysis uses **publicly available aggregate surveillance data**; no ethics approval was required.
- Indigenous health data are framed to highlight **systemic service gaps**, not population deficits.
- **Small-area suppression** is applied: counts fewer than 5 are suppressed in output tables.
- The analysis acknowledges the ongoing impacts of colonisation and systemic disadvantage on Aboriginal and Torres Strait Islander health outcomes.
